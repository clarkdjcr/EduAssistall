# Scripts Context

## Purpose
Store project utility scripts for Firebase setup, content seeding, SharePoint integration, and curriculum parsing.

## Contents
- SharePoint setup and webhook registration scripts.
- Firebase content and policy upload scripts.
- NC DPI standards parsing script.

## Conventions
- Use relative project paths inside scripts.
- Keep scripts executable or runnable from the project root.
- Prefer deterministic, repeatable transformations over manual edits.
- Log script additions or edits in `scripts/LOG.md`.
