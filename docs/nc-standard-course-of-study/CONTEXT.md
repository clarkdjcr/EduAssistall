# NC Standard Course of Study Context

## Purpose
Store local Markdown mirrors of official NC DPI Standard Course of Study pages and linked standards resources for EduAssist curriculum reference work.

## Contents
- `INDEX.md` summarizes the crawl source, subject coverage, and generated files.
- Markdown files generated from NC DPI pages and linked public Google Docs.
- `crawl_manifest.json` records source URLs, output filenames, status, and extraction notes.

## Conventions
- Preserve source URLs in each Markdown file.
- Keep filenames relative, lowercase, and stable.
- Generated files may be refreshed from the official NC DPI standards page.
- Do not treat this collection as app-bundled content unless it is explicitly added to the Xcode target.
