# Parsed NC Standards Context

## Purpose
Store generated structured records parsed from the local NC DPI Standard Course of Study Markdown mirror.

## Contents
- `standards-records.jsonl` contains one JSON record per extracted standard or document-level fallback.
- `standards-summary.json` contains parser counts by subject, grade/course, and parse quality.
- `INDEX.md` summarizes parser output and known limits.

## Conventions
- Files in this directory are generated by `scripts/parse-nc-standards.js`.
- Do not manually edit generated JSON/Markdown output; rerun the parser after source changes.
- Parser output is for review and seeding. Admin approval is still required before standards become active curriculum.
