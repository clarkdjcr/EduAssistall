# Arts Education - Course Code Crosswalk

```csv
Arts Education Course Code Crosswalk,
Text Colors,
Blue text is for new course,
Green text shows name change,
Red text is for new courses that replace the structure of retired course codes,
Comments,
Color,Meaning
No Change,No change to course name or code
Name Change,"No change to course code, only name"
New Course,New course code AND name
Remove,Course code is retired
Arts Ed Graduation Requirement & Endorsement Columns,
Color,Meaning
NA,This level does not apply to the requirements
Yes,This course satisfies the requirement
No,This course does not satisfy the requirement
Not at this time,This course does is not currently recognized as satisfying the requirement
Notes,
Remove* (with an asterisk) means that the course would count toward the graduation requirement if students took it between 22-25),
Any pertinent information regarding the courses is listed in the last column,
```
