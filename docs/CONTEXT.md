# Docs Context

## Purpose
Store non-shipping documentation, public web pages, policy references, and local support materials for EduAssist.

## Contents
- App Store and privacy documentation.
- Local marketing and support web pages.
- Seed/reference files used during development.
- `nc-standard-course-of-study/` contains local Markdown mirrors of NC DPI standards resources.

## Conventions
- Keep documentation artifacts out of the app shipping target unless explicitly added in Xcode.
- Use relative paths for project references.
- Add large or generated reference collections under a named subdirectory.
- Record file creation and edits in `LOG.md`.
