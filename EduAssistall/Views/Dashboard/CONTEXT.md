# Dashboard Views Context

## Purpose
Store role-specific dashboard screens and reusable dashboard cards for the main app tabs.

## Contents
- Teacher dashboard overview, student roster summary, quick actions, and dashboard helper views.

## Conventions
- Keep dashboard navigation labels plain and role-specific.
- Use icons that directly match the destination or action.
- Prefer concise overview cards and route deeper workflows to dedicated screens.
- Log dashboard screen additions or edits in `EduAssistall/Views/Dashboard/LOG.md`.
