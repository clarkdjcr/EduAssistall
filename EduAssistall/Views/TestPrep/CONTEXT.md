# Test Prep Views Context

## Purpose
Store student practice-test browsing, taking, and results screens.

## Contents
- Test prep catalog view.
- Practice test runner.
- Test result summary views.

## Conventions
- Keep practice tests student-facing and self-contained.
- Preserve progress/history refresh after test completion.
- Keep modal test-taking flows usable on iOS, iPadOS, and macOS.
- Log test prep view edits in `EduAssistall/Views/TestPrep/LOG.md`.
