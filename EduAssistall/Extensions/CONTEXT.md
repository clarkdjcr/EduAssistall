# Extensions Context

## Purpose
Store cross-platform SwiftUI and Foundation helpers used throughout the app.

## Contents
- Adaptive color helpers.
- Cross-platform view/input modifiers and presentation helpers.

## Conventions
- Keep platform-specific UI behavior centralized here when practical.
- Prefer small, composable view modifiers over one-off platform checks in feature views.
- Log extension edits in `EduAssistall/Extensions/LOG.md`.
